# Library-Management-System
Library Management System – A web-based application built using Java, JSP, Servlets, and MySQL that manages library operations such as adding, issuing, and returning books. Features a responsive UI, frontend validation, dynamic data rendering with JSP, and secure database integration deployed on Apache Tomcat.

This project allows librarians or administrators to manage books and users digitally, reducing manual work and improving accuracy.

Technology Stack
Frontend: HTML, CSS, JavaScript, JSP
Backend: Java Servlets
Database: MySQL (Relational Database)
Server: Apache Tomcat

Key Functionalities
Add new books to the library
View all available books
Issue books to registered users
Return books and update availability status
Maintain user and book records using relational database mapping
Frontend form validation using JavaScript
Dynamic data rendering using JSP

System Design & Implementation
Uses JDBC for secure and efficient database connectivity
Ensures data integrity and consistency using SQL constraints and relationships
Provides smooth client–server communication using Java Servlets
Features a responsive and user-friendly UI built with HTML and CSS

Database Design
Books Table: Stores book details and availability
Users Table: Stores registered user information
Transactions Table: Tracks issue and return operations

Installation & Setup
Prerequisites
Java JDK (8 or above)
Apache Tomcat (9 or 10)
MySQL Server
Eclipse IDE (Java / Java EE)

Steps:-
Import the project into Eclipse IDE as a Dynamic Web Project
Configure Apache Tomcat Server in Eclipse
Create a MySQL database and tables using the provided SQL script
Update database credentials in the JDBC configuration file
Run the project on the Tomcat server
Open the application in your browser

Deployment

Deployed and tested on Apache Tomcat Server
Works on modern web browsers
Scalable for future feature enhancements
